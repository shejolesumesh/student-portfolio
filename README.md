# Student Portfolio Website

A modern, responsive personal portfolio website built with **HTML, CSS, and JavaScript**.  
This portfolio displays my skills, projects, and contact information with a clean UI and smooth user experience.

**Live Website:**  
https://shejolesumesh.github.io/student-portfolio/

---

## Features

- Hero section with **cinematic video background**
- **Dark / Light mode** toggle
- Fully **responsive design** for mobile and desktop
- Projects section with live demos and GitHub links
- Optimized performance with fallback images
- Accessibility-friendly with ARIA labels

---

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript  
- **Styling:** Custom CSS, Google Fonts  
- **Hosting:** GitHub Pages
